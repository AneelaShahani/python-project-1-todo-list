# Python Project 1 - To-Do List Application

A simple To-Do List application developed as Project 1 of the DecodeLabs Industrial Training Kit 2026. This project is built using Python and demonstrates basic programming concepts such as lists, loops, conditionals, and file handling.

---

## 📌 Project Description

This application allows users to manage their daily tasks in a simple console-based interface. Users can add, view, delete, and mark tasks as completed.

---

## ✨ Features

- Add new tasks
- View all tasks
- Delete existing tasks
- Mark tasks as completed
- Simple and interactive command-line interface

---

## 🛠️ Technologies Used

- Python 3
- Lists
- Loops
- Conditional Statements
- File Handling (optional depending on version)

---
